Node.js é um ambiente de execução de código JavaScript de código aberto e multiplataforma que permite executar JavaScript fora do navegador. Ele é construído sobre o motor V8 do Google Chrome e permite que desenvolvedores usem JavaScript para criar aplicações do lado do servidor, ferramentas de linha de comando e muito mais. Em vez de usar threads para lidar com várias conexões simultâneas, o Node.js emprega um modelo orientado a eventos, single-threaded e não bloqueante, o que o torna eficiente em lidar com alta concorrência.
